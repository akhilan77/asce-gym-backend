# ASCE Gym Backend

This repository contains the backend server for the ASCE Gym website.

## Tech Stack
- Node.js
- Express
- MongoDB

## Setup
1. Clone the repo
2. Run `npm install`
3. Set up your `.env` file (see `.env.example`)
4. Run the server with `npm start`

## API Base URL
`https://your-deployment-url/api/`
